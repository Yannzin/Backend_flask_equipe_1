# Localização de Veículos 

Versão do Projeto didático inicial.

## Entrega 2
Descrição resumida: 
É um sistema de rastreamento de veículos, permite fazer o cadastro de veículos e 
localizá-los.


## Requisitos
- Linux (Ubuntu/Debian recomendado)
- Python 3.10
- Git (opcional, para controle de versão)


## Versão da aplicação: 1.0 


Passos para reproduzir (Rodar): 
  1. Extrair arquivo zipado 
  2. Executar no terminal o upgrade do pip: pip install --upgrade pip. 
  3. Executar o programa normalmente ou através do comando: python run app.py. 
  4. Acessar o programa através da url fornecida no terminal. 

O sistema salva o veículo com os dados válidos, (Placa, Modelo e Cidade) e faz o cadastro. 

## Ambiente: 

  - Python 3.13, Flask, VsCode.
